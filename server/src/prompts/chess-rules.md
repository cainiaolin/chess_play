# 中国象棋完整规则

## 1. 棋盘与坐标系统

- 棋盘为 9 列 x 10 行的网格
- 坐标表示：`{x, y}` 其中 x 为列(0-8)，y 为行(0-9)
- 红方在底部(y=5-9)，黑方在顶部(y=0-4)
- 楚河汉界位于第 4-5 行之间

## 3. 各棋子走法规则

### 3.1 帅/将 (k)
- **活动范围**：九宫格内（红方: x=3-5, y=7-9；黑方: x=3-5, y=0-2）
- **走法**：每步只能沿横线或竖线移动一格
- **特殊规则**：
  - 帅将不能对面（即两帅/将不能在同一列上直接照面，中间无棋子阻隔）
  - 被将死即判负

### 3.2 仕/士 (a)
- **活动范围**：九宫格内（同帅/将）
- **走法**：每步只能沿斜线移动一格（九宫格内的四个斜角点）
- **有效位置**：(3,0), (5,0), (4,1), (3,2), (5,2) 等

### 3.3 相/象 (b)
- **活动范围**：己方区域（红方: y=5-9；黑方: y=0-4），不能过河
- **走法**：沿对角线走"田"字（即移动2格对角线）
- **塞象眼**：如果"田"字中心点有棋子，则不能移动（称为塞象眼）
- **有效位置**：仅能到达棋盘上的特定点位

### 3.4 马 (n)
- **活动范围**：全棋盘
- **走法**：先横/竖走一格，再斜走一格（形成"日"字形）
- **蹩马腿**：如果横/竖方向的第一格有棋子，则不能往该方向跳（称为蹩马腿）
- **可到达位置**：从(x,y)出发，可到达8个可能位置：
  - (x+1, y-2), (x+2, y-1), (x+2, y+1), (x+1, y+2)
  - (x-1, y+2), (x-2, y+1), (x-2, y-1), (x-1, y-2)
  - 需检查边界和蹩马腿

### 3.5 车 (r)
- **活动范围**：全棋盘
- **走法**：沿横线或竖线移动任意格数
- **限制**：不能跨越其他棋子（路径上必须无障碍）

### 3.6 炮 (c)
- **活动范围**：全棋盘
- **走法**：
  - **移动**：与车相同，沿横线或竖线移动，不能跨越棋子
  - **吃子**：必须隔一个棋子（炮架）才能吃掉目标棋子
- **规则**：炮翻山（隔子打子）

### 3.7 兵/卒 (p)
- **活动范围**：
  - **过河前**：只能前进，不能横走或后退
  - **过河后**：可以前进或横走，但不能后退
- **走法**：每步只能移动一格
  - 红兵：过河前只能向 y-1 方向；过河后可 y-1 或 x±1
  - 黑卒：过河前只能向 y+1 方向；过河后可 y+1 或 x±1
- **过河定义**：红兵 y<5 为过河；黑卒 y>4 为过河

## 4. 游戏规则

### 4.1 基本规则
- 红方先走，双方轮流走棋
- 每次只能移动一个棋子
- 轮到走棋的一方，将某个棋子从一个交叉点走到另一个交叉点，或者吃掉对方的棋子而占领其位置

### 4.2 吃子规则
- 移动棋子到对方棋子所在位置，即可吃掉该棋子
- 炮吃子必须隔一个棋子

### 4.3 胜负判定
- **将死**：一方的帅/将被对方将军，且无法解除将军状态
- **困毙**：轮到走棋的一方无子可走（所有棋子都无法合法移动）
- **认输**：一方主动认输

### 4.4 和棋情况
- 双方同意和棋
- 出现三次重复局面
- 连续60回合无吃子
- 双方都无取胜可能

### 4.5 禁止着法
- **将帅对面**：帅/将不能在同一列上无阻挡地对面
- **自杀**：不能走出让己方帅/将被将军的棋
- **长将**：不能连续将军超过允许次数

## 6. 输出格式要求

你的每次走棋决策必须严格按照以下 JSON 格式输出：

```json
{
  "from": {"x": <列号0-8>, "y": <行号0-9>},
  "to": {"x": <列号0-8>, "y": <行号0-9>},
  "thinking": "<简要说明你的思考过程，包括：为何选择这个棋子、为何选择这个目标位置、考虑了哪些威胁或机会>"
}
```

### 输出要求：
1. **必须**返回合法的 JSON 格式
2. **from** 和 **to** 必须是有效的棋盘坐标
3. 走法必须符合上述棋子移动规则
4. 不能走出让己方被将军的棋
5. thinking 字段简要说明决策理由

### 坐标示例：
- 红方帅的初始位置：`{"x": 4, "y": 9}`
- 黑方将的初始位置：`{"x": 4, "y": 0}`
- 红方右炮的初始位置：`{"x": 7, "y": 7}`
